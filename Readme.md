# 🌌 NASA Mission Explorer

## 📌 Project Overview

NASA Mission Explorer is a web application that allows users to explore NASA’s "Astronomy Picture of the Day" (APOD). Users can select any date to view space images along with detailed explanations provided by NASA. This project provides a visually engaging way to explore the universe.

## 🎯 Purpose

The purpose of this project is to build a dynamic web application using a public API. It focuses on learning API integration, handling user input, and designing an interactive user interface with features like search (by date), filtering, and content display.

## 🔌 API Used

This project uses the NASA Open API (Astronomy Picture of the Day - APOD).

Data fetched includes:

* Astronomy images
* Titles of the images
* Descriptions/explanations
* Dates of publication
* High-definition (HD) images

API Link: https://api.nasa.gov/

## ✨ Features (Planned)

* 📅 Select a date to view past APOD data
* 🌌 Display astronomy image with title and description
* 🔍 Search by date (e.g., birthday search feature)
* 🖼️ Full-screen image modal view
* 🔘 Toggle between standard and HD image quality
* 🎨 Dark space-themed user interface
